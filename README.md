# Lineup creator for War Thunder
If you`ve ever organized events in War Thunder with special lineups, you know how hard vehicle lineup creation is. The main task of this program is to help you with vehicle lineup creation.
> NOTE: This program downloads images from the [official War Thunder wiki](https://wiki.warthunder.com/Main_Page), so stay connected!
<br>

---

## How to use
### Toolbox menu

![image](https://github.com/Gaz1zPr0g/wt-lineup-creator/assets/81079946/645cf554-8c09-4bda-ad6e-cedf2296aeb3)

#### File
`Save` or `CTRL+S` saves .json schemes to load them later. \
`Import` or `CTRL+O` opens .json schemes. \
`Save as...` or `CTRL+SHIFT+S` saves .png image with table. 

`Report bug` opens [issue page](https://github.com/Gaz1zPr0g/wt-lineup-creator/issues) \
`Support` opens [Boosty](https://boosty.to/railguntoaster) \
`Language` combobox changes language 
<br>
#### Export settings
`Vehicle types order` allows change the order of the vehicles on the image \
`Background` image background color

#### Country box and status bars
`Country` combobox selects the templates and changes the flag (if unlocked) \
`flag locked` or `flag unlocked` locks or unlocks flag for country combobox changes \
`Edit mode` - editing data in cells \
`Delete mode` - deleteng data in cells 

---

### Tables and cells
![image](https://github.com/Gaz1zPr0g/wt-lineup-creator/assets/81079946/2638c7ec-ce16-4c5b-a0ba-e72d28e282f6)

There are 5 tables - Tanks, planes, helicopters, coastal fleet, bluewater fleet \

Right click on `Add` button to add row \
Left click on `Add` button to add column

Left click on the vehicle cell to open vehicle data editing screen
---
![image](https://github.com/Gaz1zPr0g/wt-lineup-creator/assets/81079946/596a5fb1-722f-4952-9f71-f108460dc39f)

`Vehicle name` - templates by name
`Vehicle text` - text on the cell
`Vehicle icon` - all in-game vehicle icons
`Background` - blue, golden or green
